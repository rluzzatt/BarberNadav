# נדב פרץ

אתר סטטי לנדב פרץ, בעברית טבעית ופשוטה, עם דגש על תספורות גברים, פיידים, עיצוב זקן וקביעת תור.

## פתיחה מקומית

אפשר לפתוח את `index.html` ישירות בדפדפן, או להריץ:

```powershell
python -m http.server 5173
```

ואז להיכנס ל־`http://localhost:5173`.

## התאמות המשך

- להחליף את `nadav@example.com` ב־`index.html` לאימייל, וואטסאפ או מערכת תורים אמיתית.
- לעדכן שמות שירותים, זמני שירות ומחירים כשיהיה תפריט סופי.
- להוסיף כתובת, שעות פעילות, אינסטגרם וטלפון באזור הביקור.
- התמונות הנוכחיות נמצאות ב־`assets/nadav-background.jpg` ו־`assets/nadav-photo.jpg`.
